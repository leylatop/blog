http（Hyper Text Transfer Protocol）：超文本传输协议

- 基于 TCP/IP 协议
- **应用层协议**
- 服务器传输超文本到本地浏览器的传送协议
- 默认使用80端口
- 规定了客户端和服务器之间的通信格式
	- 请求命令：get/post/delete/put/head
	- 请求内容和格式：文本、图片、视频、二进制文件
	- 请求头
	- 请求体
	- 响应头
	- 响应体
	- 状态码
	- 多字符集
	- 多部分发送
	- 权限
	- 缓存
	- 内容编码
- 版本：
	- http 0.9
	- http 1.0
	- http 1.1
	- http 2
	- http 3
