**Node.js 和 npm**
   - **http-server**: 一个简单的命令行 HTTP 服务器
 ```bash
 npm install -g http-server
 http-server
 ```
   - **live-server**: 类似于 http-server，但支持自动刷新浏览器
 ```bash
 npm install -g live-server
 live-server
     ```
