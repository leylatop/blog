title: BOM和DOM的关系
---
包含关系，DOM 是 BOM 的成员之一
名词解释：
- BOM（brower object model）：浏览器对象模型
- DOM（document object model）：文档对象模型

![图片](http://images.leyla.top/note/Pastedimage20240621093711.png)

BOM 包括以下内容：

- document (DOM)
- frames
- history
- location
- navigator
- screen
